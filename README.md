# Анализ данных, регрессия и генерация сочетаний

Материалы научно-исследовательской работы (1-й курс) в РТУ МИРЭА. Включает три части: регрессионный анализ в R, классификацию в Python и реализацию алгоритма генерации сочетаний на C++.

## Части

### 1. Регрессионный анализ (R)
- Набор данных `swiss`.
- Построение простых и множественных линейных моделей для переменных `Agriculture` и `Examination`.
- Проверка мультиколлинеарности, добавление логарифмов и произведений признаков.
- Оценка моделей по R², p-значениям и доверительным интервалам.

### 2. Классификация (Python)
- Датасеты: [HR Analytics: Job Change](https://www.kaggle.com/arashnic/hr-analytics-job-change-of-data-scientists) и [House Sales in King County](https://www.kaggle.com/harlfoxem/housesalesprediction).
- Предобработка (пропуски, кодирование, нормализация).
- Построение логистической регрессии и случайного леса.
- Подбор гиперпараметров через GridSearchCV.
- На классификации спален (King County) достигнуты precision и F1 ~75%.

### 3. Генерация сочетаний (C++)
- Реализован алгоритм 382 (Phillip J. Chase) для генерации всех сочетаний из N по M без повторений.
- Сложность O(C(N,M)), проведены замеры времени.
- Применение: комбинаторная оптимизация, тестирование.

## Технологии
- **R**: lm, car, lmtest, dplyr
- **Python**: pandas, numpy, scikit-learn, matplotlib
- **C++**: STL, iostream, vector

## Запуск
- R-скрипты: открыть `regression_swiss.R` в RStudio.
- Python-ноутбуки: `classification_kingcounty.ipynb`, `hr_analytics.ipynb`
- C++: `g++ combinations.cpp -o combinations && ./combinations`

## Результаты
- В регрессии лучшая модель для `Examination` включала `log(Fertility)` и `Catholic` (R² ≈ 0.46).
- На King County: логистическая регрессия F1 ≈ 0.69, случайный лес F1 ≈ 0.74.
- Алгоритм сочетаний протестирован для N до 20; рост времени соответствует биномиальному коэффициенту.
